# Phase 71-01: Base UI - Buttons & Inputs

## Objective

Create reusable Button, Input, and Textarea components that use the CSS custom properties from Phase 70, ensuring consistent synthwave styling across the application.

## Reference

From `doc/1.png`:
- Primary button: Cyan background with dark text, rounded corners
- Secondary button: Magenta outline, transparent fill
- Input fields: Dark background with magenta border, placeholder text
- Neon glow effects on focus/hover

## Tasks

### Task 1: Button Component

Create `src/renderer/src/components/UI/Button.tsx`:

**Variants:**
- `primary` - Cyan background, dark text, cyan glow on hover
- `secondary` - Transparent with magenta border, magenta glow on hover
- `ghost` - Transparent, subtle hover background
- `danger` - Red/error variant for destructive actions

**Sizes:**
- `sm` - Small padding, smaller text
- `md` - Default size
- `lg` - Large padding, larger text

**States:**
- Default, hover, focus, active, disabled, loading

**Props:**
```typescript
interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'secondary' | 'ghost' | 'danger';
  size?: 'sm' | 'md' | 'lg';
  loading?: boolean;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
}
```

### Task 2: IconButton Component

Create `src/renderer/src/components/UI/IconButton.tsx`:

A square button for icon-only actions (close, settings, etc.)

**Props:**
```typescript
interface IconButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'secondary' | 'ghost';
  size?: 'sm' | 'md' | 'lg';
  label: string; // Required for accessibility (aria-label)
  children: React.ReactNode; // The icon
}
```

### Task 3: Input Component

Create `src/renderer/src/components/UI/Input.tsx`:

**Types:**
- `text` - Standard text input
- `number` - Number input with optional min/max
- `password` - Password with optional show/hide toggle
- `search` - Search input with clear button

**States:**
- Default, focus, error, disabled

**Props:**
```typescript
interface InputProps extends Omit<React.InputHTMLAttributes<HTMLInputElement>, 'size'> {
  size?: 'sm' | 'md' | 'lg';
  error?: boolean;
  errorMessage?: string;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
}
```

### Task 4: Textarea Component

Create `src/renderer/src/components/UI/Textarea.tsx`:

**Features:**
- Auto-resize option (grows with content)
- Min/max rows
- Character count (optional)

**Props:**
```typescript
interface TextareaProps extends React.TextareaHTMLAttributes<HTMLTextAreaElement> {
  size?: 'sm' | 'md' | 'lg';
  error?: boolean;
  errorMessage?: string;
  autoResize?: boolean;
  minRows?: number;
  maxRows?: number;
  showCount?: boolean;
}
```

### Task 5: Barrel Export

Create `src/renderer/src/components/UI/index.ts`:

Export all UI components for easy imports.

## CSS Strategy

Components use CSS custom properties from `synthwave.css`:

```css
/* Button example */
.ui-button-primary {
  background: var(--btn-primary-bg);
  color: var(--btn-primary-text);
  border: 1px solid var(--btn-primary-border);
  transition: all var(--transition-fast);
}

.ui-button-primary:hover {
  background: var(--btn-primary-hover-bg);
  box-shadow: var(--btn-primary-glow);
}
```

Use Tailwind only for:
- Layout (flex, grid, positioning)
- Sizing (w-full, h-10)
- Spacing (px-4, py-2, gap-2)
- Display utilities
