<task type="auto">
  <name>Task 1: Create global scrollbar CSS</name>
  <files>src/renderer/src/styles/scrollbar.css</files>
  <action>
    Create new file `src/renderer/src/styles/scrollbar.css` with comprehensive scrollbar styling:

    **Webkit scrollbars** (Chrome, Edge, Safari):
    ```css
    /* Scrollbar width/height */
    ::-webkit-scrollbar {
      width: 8px;
      height: 8px;
    }

    /* Scrollbar track (background) */
    ::-webkit-scrollbar-track {
      background: rgba(0, 0, 0, 0.2);
      border-radius: 4px;
    }

    /* Scrollbar thumb (draggable part) */
    ::-webkit-scrollbar-thumb {
      background: rgba(168, 108, 230, 0.4); /* Purple accent with transparency */
      border-radius: 4px;
      transition: background 0.2s ease;
    }

    /* Scrollbar thumb hover */
    ::-webkit-scrollbar-thumb:hover {
      background: rgba(168, 108, 230, 0.7); /* Brighter purple on hover */
    }

    /* Scrollbar thumb active (being dragged) */
    ::-webkit-scrollbar-thumb:active {
      background: rgba(168, 108, 230, 0.9); /* Full purple when dragging */
    }

    /* Corner (where horizontal and vertical scrollbars meet) */
    ::-webkit-scrollbar-corner {
      background: rgba(0, 0, 0, 0.2);
    }
    ```

    **Firefox scrollbars**:
    ```css
    /* Firefox scrollbar styling */
    * {
      scrollbar-width: thin;
      scrollbar-color: rgba(168, 108, 230, 0.4) rgba(0, 0, 0, 0.2);
    }
    ```

    **Color constants** (match synthwave theme):
    - Purple accent: #a86ce6 (168, 108, 230 in RGB)
    - Cyan accent (alternative): #00f0ff (0, 240, 255 in RGB)
    - Use purple as primary, cyan as optional variant

    **Variants for different contexts**:
    - Add .scrollbar-cyan class for cyan scrollbars (optional):
      ```css
      .scrollbar-cyan::-webkit-scrollbar-thumb {
        background: rgba(0, 240, 255, 0.4);
      }
      .scrollbar-cyan::-webkit-scrollbar-thumb:hover {
        background: rgba(0, 240, 255, 0.7);
      }
      ```

    Why: Consistent synthwave scrollbars enhance visual cohesion across the app.
    Avoid: Do not make scrollbars too thick (8px is good balance).
  </action>
  <verify>CSS file created with no syntax errors</verify>
  <done>scrollbar.css file exists with webkit and Firefox scrollbar styles</done>
